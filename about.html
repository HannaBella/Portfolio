Collaborative Projects

Cinephile Central

Live Demo Github

Cinephile Central was created to serve as a one-stop shop for all movie fans. Users can discover an endless variety of movies and reviews from top critics. Cinephile Central also streamlines the process of finding movies, by providing video links.

I built landing page, movie-review feature and detail page. Movie-review feature includes searching a movie review by its title, calling NY Times API and rendering search result to the UI. I also integrated OMDB API to get imdb id that was used to retrieve streaming sites.

Cinephile Cental is a front end app that was built by using HTML5, Bootstrap, CSS3, JavaScript, jQuery, Lodash and API call to NY Times, OMDB, Utelly, smartip & videospider.

chefMEup

Live Demo Github

chefMEup is created to render recipes based on user's ingredients. It helps user to cook a simple, yet delicious, meal. Users can quickly mark the ingredients they have in their fridge and have access to all the recipes in our database that incorporate those ingredients. When user sign in, they can save their favorite recipes and they can even add their own recipe so that chefMEup users can use it.

I was responsible for database design, association and query. Built ingredients and recipe api.This full stack app was built by using HTML5, Bootstrap, CSS3, JavaScript, Passport, handlebars, MySQL, Sequelize, NodeJs, Express, and Heroku.

Mheels

Live Demo Github

Mheels is an app that targets food vendors who are on the road, either by pushing a cart or food trucks. The app provides vendors the ability to show their location so that customers can easily track where they are and see a menu with prices and description of the items they sell. As a customer, Mheels lets users to search for vendors depending on the type of food they are looking for around their location. Vendor can tweet about their business hours, events and the like through Mheels social media feature.

I Built most of the components and the general layout. Developed client side API to consume server side API. Mheels is a full stack app was built by using MERN(MongoDB, Express, React, Node) stack, ES6, Reactstrap, JavaScript, Auth0, Google API, Twitter API and Heroku.
